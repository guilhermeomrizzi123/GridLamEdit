Agora vou precisar andar Pesquisar Criar uma apresentação Para explicar o funcionamento desse Executava Olha eu sou de criação de laminados Lucatia Essa execução de Lernon ela esse programa É a função de editar laminados Ao qual importantamos do catia V5 da ferramenta grid design, Nessa ferramenta de grid design Através dela Construímos todas as células Necessárias Para que possamos associar O laminado E cada uma das células E a associação do laminado Pode ser feita Através Exportação E importação Tinha uma planil Tchau Tchau com Deus ha Excel Qual formato padronizado Ao qual A ferramenta entende O padrão dos dados Dificuldades D utilizar o Excel Para editar Esses laminado É muito grande pois Para construção dos laminados precisamos copiar os dados Gerados pela Engenharia de análise estrutural E posteriormente Organizar esses dados Na vertical e também fazer a simetria desses dados Em ordem decrescente e adicionar os dados gerados Na Mesma coluna Dessa forma é possível ir construindo os laminados, que no caso do exemplo do lower skin o grid design tem aproximadamente 168 celulas e para cada celula se deve associar um laminado as vezes pode repetir o mesmo laminado para celulas diferente mas nem sempre, entao as dificuldades sao essas, trabalhar com o excel para criar e manipular esses laminados é algo cansativo, dificil e as chances de errro sao grandes, principalmente pois ficar verificando manualmente se o laminado é simetrico é algo muito chato e passivel de erro.
Eu desenvolvi essa ferramenta para a edicao da planilha que exportamos do grid design do catia v5 para auxiliar na edicao da planilha excel, o processo de utilizacao dessa planilha se inicia após a construcao das celulas do laminado nocatia v5 e apos a definicao das celulas geramos o grid design e exportamos a planilha do grid design, apos essa exportacao podemos carregar a planilha na ferramenta que eu criei e denominei GridLamEdit, quando carregamos a planilha nesse executável podemos ver todas as celulas e quais laminados estao associados a cada celula, ao clicar na celula desejada podemos ver o laminado e as ferramentas para edicao, criacao, manipulacao, verificacao de simetria todas ficam abertas para que o laminado seja editado, entao essa ferramenta exclui a necessidade de ficar editando o laminado dentro da planilha excel. Tambem conta com um recurso muito util para criacao de novos laminados partindo de informacoes de direcao de camadas fornecidas pela engenhaira de analise estrutural, onde podemos copiar uma sequencia de direcoes de camadas e se desejar que essa sequencia seja simetrica tambem podemos selecionar essa opcao, assim dessa forma podemos criar um novo laminado stacking partindo diretamente apenas das informacoes fornecidas pela engenharia de analise estrutural. Apos as edicoes serem feitas, podemos salvar uma versao do programa para continuar a edição posteriormente ou podemos ir direto para exportar a planilha excel com as novas alteracoes as quais fizemos utilizando o programa, ao clicar em exportar abre uma caixa de Relatorio de verificacao de laminados, onde é possivel ver um resumo dos laminados simetricos e nao simetricos e tambem os laminados duplicados se caso existirem, tambem existe a opcao de deletar laminados duplicados caso esses nao estejam associados a nenhuma celula. apos essas verificacoes o usuario podera exportar a planilha com todas as alteracoes feitas usando o programa.
